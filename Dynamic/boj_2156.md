# 백준 2156 포도주 시식

### 마지막에 max(DP) 를 출력해야 하는 이유 다시 한 번 상기
```python
import sys
input = sys.stdin.readline


n = int(input())
l = [0] * 10001
for i in range(1, n+1):
    l[i] = int(input())

DP = [0] * 10001
DP[1], DP[2] = l[1], l[1] + l[2]

for i in range(3, n+1):
    DP[i] = max(DP[i-1], l[i] + DP[i-2], l[i]+l[i-1]+DP[i-3])
    
print(max(DP))
```
