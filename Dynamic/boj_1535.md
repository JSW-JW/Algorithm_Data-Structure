# 백준 1535 포도주 시식

```python
import sys
input = sys.stdin.readline


n = int(input())
l = [0] * 10001
for i in range(1, n+1):
    l[i] = int(input())

DP = [0] * 10001
# DP[N] = N 번째 잔까지 연속 3잔 미만으로 마실 수 있는 최대량
DP[1], DP[2] = l[1], l[1] + l[2]

# 6, 10, 13, 9, 8, 1 
for i in range(3, n+1):
    DP[i] = max(DP[i-1], l[i] + DP[i-2], l[i]+l[i-1]+DP[i-3])
    
print(max(DP))
```
